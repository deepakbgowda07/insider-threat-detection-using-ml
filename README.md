# Real-Time Insider Threat Detection System (V2)

## Overview
This project detects insider threats using real-time user activity logs and unsupervised machine learning.

The system collects live logs, extracts behavioral features, detects anomalies, and visualizes threats in real time.

## Team Split
- Developer A: Real-time systems, log collection, streaming, security
- Developer B: Feature engineering, ML models, visualization, analysis

## Tech Stack
- Python
- Kafka / Streaming
- Scikit-learn
- TensorFlow
- Streamlit
